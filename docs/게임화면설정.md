# 게임 화면 설정 사양서

## 배경 시스템

### 스테이지별 배경 테마

#### 스테이지 1: 초원 & 산 (🌿🏔️)
- 상단: 파란 하늘
- 중단: 녹색 초원
- 배경 요소: 산 실루엣

#### 스테이지 2: 협곡 (🏜️)
- 상단: 옅은 하늘
- 중단: 황갈색 협곡
- 배경 요소: 절벽 실루엣

#### 스테이지 3: 사막 (🏜️)
- 상단: 뜨거운 하늘 (주황빛)
- 중단: 모래 사막 (노란색)
- 배경 요소: 모래 언덕

#### 스테이지 4: 바다 (🌊)
- 상단: 밝은 파란 하늘
- 중단: 깊은 바다 (파란색)
- 배경 요소: 파도 효과

#### 스테이지 5: 화산 (🌋)
- 상단: 어두운 연기 (회색)
- 중단: 용암 (빨간색/주황색)
- 배경 요소: 화산 분화구

#### 스테이지 6: 얼음 성 (❄️🏰)
- 상단: 차가운 하늘 (연한 파랑)
- 중단: 얼음 지대 (흰색/청록색)
- 배경 요소: 얼음 결정

#### 스테이지 7: 천국 (☁️✨)
- 상단: 황금빛 하늘
- 중단: 구름 층 (흰색)
- 배경 요소: 빛나는 효과

#### 스테이지 8: 지옥 (🔥💀)
- 상단: 어두운 붉은 하늘
- 중단: 불타는 지옥 (검은색/빨간색)
- 배경 요소: 불길 효과

## 몬스터 비주얼

### 일반 몬스터
- 모양: 원형
- 반지름: 16px
- 색상: 빨간색 (0xFFFF5555)
- 이모지 표시: 👾

### 미니보스 (라운드 5)
- 모양: 원형 (크기 동일)
- 색상: 보라색 (0xFF9C27B0)
- 이모지 표시: 😈 (악마)
- HP 바: 더 두껍게 표시

### 보스 (라운드 10)
- 모양: 원형 (크기 동일)
- 색상: 검은색 (0xFF212121)
- 이모지 표시: 😈 (악마)
- HP 바: 가장 두껍게 표시

### 몬스터 HP 바
- 위치: 몬스터 위쪽
- 크기: 몬스터 직경 × 4px
- 배경: 회색 (0xFF666666)
- 전경: 빨간색 (0xFFFF0000)
- HP 비율에 따라 길이 변경

## 성(Castle) 비주얼

### 기본 외형
- 높이: 80px
- 위치: 화면 하단 중앙
- 모양: 사각형 기반
- 색상: 갈색 계열 (0xFF8D6E63)
- 이모지: 🏰

### 성 HP 바
- 위치: 성 상단
- 너비: 성 너비와 동일
- 높이: 8px
- 배경: 어두운 회색
- 전경: 초록색 (HP 높음) → 노란색 (HP 중간) → 빨간색 (HP 낮음)

## UI 요소 스타일

### 버튼 스타일

#### 기본 버튼
- 배경: 파란색 (0xFF2196F3)
- 텍스트: 흰색
- 모서리: 둥근 사각형 (radius: 8px)
- 그림자: 약간의 드롭 섀도우

#### 비활성화 버튼
- 배경: 회색 (0xFF9E9E9E)
- 텍스트: 어두운 회색 (0xFF616161)

#### 뽑기 버튼
- 단일 소환: 파란색 배경, 작은 크기
- 10연차 소환: 보라색 배경, 큰 크기
- 젬 부족 시: 회색으로 변경

### 텍스트 스타일

#### 제목 텍스트
- 폰트 크기: 24px
- 색상: 검정 (0xFF000000)
- 굵기: 굵게 (bold)

#### 본문 텍스트
- 폰트 크기: 14px
- 색상: 어두운 회색 (0xFF424242)
- 굵기: 보통 (normal)

#### 강조 텍스트
- 폰트 크기: 18px
- 색상: 파란색 또는 금색
- 굵기: 굵게 (bold)

### 라운드 노드 스타일

#### 언락된 라운드
- 배경: 흰색 (0xFFFFFFFF)
- 테두리: 파란색 (0xFF2196F3), 2px
- 텍스트: 검정

#### 언락되지 않은 라운드
- 배경: 회색 (0xFFBDBDBD)
- 테두리: 어두운 회색 (0xFF757575), 2px
- 아이콘: 🔒
- 텍스트: 회색

#### 현재 선택된 라운드
- 배경: 파란색 (0xFF2196F3)
- 테두리: 진한 파란색 (0xFF1976D2), 3px
- 텍스트: 흰색

#### 미니보스/보스 라운드
- 추가 표시: 😈 이모지
- 테두리: 보라색 (미니보스), 빨간색 (보스)

## 애니메이션 효과

### 로딩 게이지
- 위치: 화면 중앙
- 스타일: 가로 프로그레스 바
- 색상: 파란색 → 초록색 그라데이션
- 지속 시간: 0.5초

### 몬스터 낙하 애니메이션
- 시작 위치: 화면 상단 (y=0)
- 종료 위치: 지면 (y=400)
- 속도: 80px/초
- 이징: 선형 (linear)

### 몬스터 걷기 애니메이션
- 이동 방향: 아래쪽 (성을 향해)
- 속도: 50px/초
- 이징: 선형 (linear)

### 라운드 클리어 애니메이션
- 효과: "라운드 클리어!" 텍스트 페이드 인
- 지속 시간: 2초
- 배경: 반투명 흰색 오버레이

### 뽑기 결과 애니메이션
- 카드 등장: 페이드 인 + 약간의 스케일 업
- 랭크 표시: 반짝이는 효과 (S/A 랭크)
- 지속 시간: 캐릭터당 1초

## 게임 플레이 화면 상세

### 남은 몬스터 카운터
```
┌──────────────────┐
│ 남은 몬스터: 15  │
└──────────────────┘
```
- 위치: 좌측 상단 (전투 화면)
- 배경: 반투명 검정 (0x88000000)
- 텍스트: 흰색
- 실시간 업데이트

### 시간 표시
```
┌──────────────────┐
│  02:35 / 05:00   │
└──────────────────┘
```
- 위치: 우측 상단 (전투 화면)
- 형식: {경과시간} / {제한시간}
- 색상: 흰색 (정상) → 노란색 (30초 미만) → 빨간색 (10초 미만)
- 초 단위 카운트다운

### 일시정지 버튼
```
┌────┐
│ ⏸ │
└────┘
```
- 위치: 스테이지-라운드 표시 우측
- 크기: 30px × 30px (작은 버튼)
- 배경: 반투명 흰색
- 아이콘: ⏸ (일시정지)

### 캐릭터 슬롯
```
┌────┐ ┌────┐ ┌────┐ ┌────┐
│ 1  │ │ 2  │ │ 3  │ │ 4  │
└────┘ └────┘ └────┘ └────┘
```
- 위치: 성 위쪽
- 개수: 4개
- 간격: 10px
- 크기: 50px × 50px
- 배경: 회색 (비어있음), 파란색 (캐릭터 배치됨)
