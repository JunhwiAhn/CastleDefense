## 1. 概要

バフカード8種 + 属性バフ6種（計14種）とオーグメント34種（Common14/Rare12/Legendary8）について、実装コードと仕様書を照合し、以下の観点で体感評価を行った。

- **体感の実感度**: 効果が数値的・視覚的にプレイヤーに伝わるか
- **選択の面白さ**: 3択で悩む場面が生まれるか、明確な最適解が存在しないか
- **名前と効果の一致度**: ネーミングがゲーム内効果を正しく伝えているか
- **役割別適性**: クラスによって価値が変わるバフが存在するか
- **ハズレ問題**: 取っても効果を実感できない「死にバフ」がないか

**総合評価**: バフカード・増強ともに設計は堅実で、Vampire Survivors系の「少しずつ強くなる快感」と「ビルド選択の面白さ」を両立できている。ただし、いくつかの増強に体感しにくい効果や、明らかな強弱格差があるため、調整提案をまとめた。

## 2. バフカード個別評価

### 2.1 基本バフ8種

| # | バフ名 | 体感実感度 | 選択価値 | 評価 |
|---|--------|-----------|---------|------|
| 1 | 攻撃力UP (+15%) | A | A | 乗算累積で5回目に2倍。DPS上昇が敵の溶け具合で明確に体感できる。全クラスに有用 |
| 2 | 攻撃速度UP (-10%間隔) | A | A | 弾幕密度の変化で視覚的にも分かりやすい。총잡이メインでは特に爽快感が高い |
| 3 | 移動速度UP (+20%) | B+ | B | 3回上限で最大1.73倍。スティック操作時に明確に速く感じる。ただし回避能力が重要な局面が限定的（敵の接触ダメージが0.5秒毎1と低い）なため、攻撃系と比べて選択優先度が下がりがち |
| 4 | 射程UP (+15%) | B | B- | 遠距離クラス（마법사/총잡이）では恩恵が大きいが、물리딜러では射程525px→570pxの差が体感しにくい。3回上限も他バフより早く到達する |
| 5 | 城修復 (HP+20) | A | A | 城HP200に対して20回復は10%であり、即効性のある緊急回避手段として常に選択肢に入る。無制限重複も適切 |
| 6 | タワー強化 (+10%) | B | B+ | 5回で1.61倍は十分だが、タワーの攻撃が「プレイヤーの操作」ではないため、自分が強くなった実感が薄い。ただしタワー4基の累積DPSは大きいため、実質的な効果は高い |
| 7 | XP磁石拡大 (+15px) | B- | C+ | 基本20px→3回で65pxは有用だが、「便利」であり「強い」わけではない。後半ラウンドでXPが大量に散乱する場面では価値が上がるが、攻撃/城修復と比較すると優先度が低い |
| 8 | 城バリア (10秒無敵) | A | A | ボスラウンドでの緊急防御として非常に強力。視覚的にもバリアVFXが明確で、取得時の満足感が高い |

**バフカード8種の体感総評**:

- **明確に体感できる**: 攻撃力UP、攻撃速度UP、城修復、城バリア（4種）
- **条件付きで体感**: 移動速度UP、タワー強化、射程UP（3種）
- **体感が薄い場面あり**: XP磁石拡大（1種）

3択構成の推奨ロジック（メイン強化系1 + 城系1 + ユーティリティ1）は適切に機能しており、「攻撃を取るか、城を守るか」の二択的な判断が頻繁に発生する。この構造は選択の面白さとして合格。

### 2.2 属性バフ6種

| # | バフ名 | 体感実感度 | 選択価値 | 評価 |
|---|--------|-----------|---------|------|
| 1-5 | 火/水/地/電気/闇 属性付与 | B+ | A- | 属性相性（有利x1.5/不利x0.75）が直接ダメージに反映されるため、有利属性の敵に対しては明確に体感できる。ただしステージ別の敵属性を事前に知らないと「どの属性を取るべきか」の判断が難しい |
| 6 | 属性マスタリー (+10%) | B- | C+ | +10%の追加ダメージは微量で体感しにくい。3回取得しても+30%で、属性有利（x1.5）と比べるとインパクトが小さい |

**属性バフの体感総評**:

- 属性付与バフは「上書き式」（1つしか持てない）のため、後から別の属性を取ると前の属性が消える。この仕様はプレイヤーに分かりにくく、説明が必要。
- 属性マスタリーは属性付与後でないと意味がなく、属性なしの状態で出現すると死にバフになる。出現条件に「属性付与済み」を追加すべき。
- ステージの敵属性構成が見える場合、有利属性を選ぶ戦略性が生まれる。見えない場合は「闇（全属性x1.1）」が無難な選択肢になりがち。

## 4. 選択の面白さ分析

### 4.1 バフカード3択の悩みポイント

以下のような場面で「悩む選択」が成立する:

1. **攻撃力UP vs 城修復**: 城HPが減っている時に攻撃を伸ばすか守りを固めるか
2. **攻撃速度UP vs タワー強化**: メインキャラを強化するか、タワー全体を底上げするか
3. **城バリア vs 攻撃系**: 10秒無敵の安心感か、恒久的な火力向上か
4. **属性付与 vs ステータス強化**: 属性相性を活かすか、純粋な数値を上げるか

**問題点**:
- XP磁石拡大は他のどのバフと並んでも選択優先度が低い。「便利だが弱い」ポジションが固定されてしまう。
- 属性マスタリーが属性未取得時に出現すると完全なハズレ。抽選ロジックで除外すべき。
- 射程UPが3回上限に早く達するため、中盤以降の出現が無駄になりやすい。

### 4.2 増強3択の悩みポイント

1. **R-01連鎖弾 vs R-02吸血衝動**: 攻撃力拡張か生存力か
2. **R-06緊急発動 vs R-03爆発弾頭**: 必殺技頻度か通常火力か
3. **L-06究極砲台 vs L-07魂の連鎖**: タワー単体強化かメイン-タワーシナジーか

**問題点**:
- ティア混合の3択（Common/Rare/Legendary）では、Legendaryが出た時にほぼ確実にLegendaryを選ぶ。ティア間の選択に悩みが生まれにくい。
- Commonの弱い増強（C-08/C-11/C-13/C-14）は、同ティア内でも選択されにくい。

### 4.3 組み合わせシナジー

以下の組み合わせが面白い相乗効果を生む:

| 組み合わせ | シナジー効果 |
|-----------|------------|
| L-07魂の連鎖 + L-06究極砲台 | メイン撃破でタワーのクリティカル弾が+1発 |
| R-01連鎖弾 + C-10余波 | 撃破時のスプラッシュ+連鎖で範囲殲滅力大幅UP |
| L-02王の咆哮 + R-06緊急発動 | 必殺技の頻度UP+必殺技後のタワー超強化 |
| R-02吸血衝動 + C-03鋭利な刃 | 攻撃力UP→吸血量もUP |
| L-08永遠の契約 + バフ攻撃力UP×2 | バフ選択時に追加スタックでDPS急上昇 |

組み合わせの面白さは十分にあり、「ビルドを考える楽しさ」が増強システムの核心的な価値として機能している。

## 5. 調整提案

### 優先度: 高（バグ修正）

| # | 対象 | 問題 | 提案 |
|---|------|------|------|
| 1 | R-08 鉄壁の城 | 通常モンスター接触ダメージ1×0.7=0（切り捨て）で完全無効化 | `max(1, (damage * 0.7).floor())` に統一。line 1180の処理を修正 |
| 2 | C-12 タワー補給 | `_rangeUpCount`を増加させており、メインキャラの射程も上がる | タワー専用の射程変数を追加するか、仕様をメイン+タワー射程UPに変更 |
| 3 | C-06 タワー油断 | `_towerUpCount`を増加させており、タワー攻撃力UPと同一処理。名前が「攻速+20%」と異なる | タワー攻速専用変数を追加するか、名前を「タワー強化」に変更 |

### 優先度: 中（バランス調整）

| # | 対象 | 問題 | 提案 |
|---|------|------|------|
| 4 | L-04 時の加速 | 3秒インターバルでのみ発動。Legendary感が薄い | 効果を拡張: 「インターバル中+ラウンド開始後5秒間も移速200%/XP磁石発動」とし、ラウンド序盤の立ち上がりにも恩恵を与える |
| 5 | C-11 守護の炎 | ラウンド開始時城HP+5は微量すぎる | HP+5→HP+15に引き上げ。5ラウンドで+75なら城HP200に対して約37%で体感可能 |
| 6 | C-08/C-13 復活系 | 死亡頻度が低いと恩恵ゼロ | C-08に追加効果「復活時HP+10」、C-13に追加効果「無敵中に攻撃力+30%」を付与し、発動時のリターンを大きくする |
| 7 | 属性マスタリー/C-14 | 属性未取得時に効果ゼロ | 抽選ロジックで属性未取得時は出現しないよう除外条件を追加 |
| 8 | R-09/R-10 | Rare同士の比較で選択されにくい | R-09に追加効果「消滅直前のXP/ゴールドが2倍の価値」、R-10に追加効果「2回目のマグネット範囲が1.5倍」 |

### 優先度: 低（体験向上）

| # | 対象 | 問題 | 提案 |
|---|------|------|------|
| 9 | 属性付与バフ | 上書き式が分かりにくい | 属性変更時に確認ダイアログ「現在の属性(火)を(水)に変更しますか?」を追加 |
| 10 | L-05 元素の嵐 | チーム属性3以上の条件が厳しい | 条件未達成時は提示しない。または条件を「2属性以上」に緩和 |
| 11 | バフカード全般 | 上限到達後の表示 | 上限到達バフにMAXマーク表示。現在のスタック数を常時HUDに表示（「ATK x3/5」等） |
| 12 | 増強カード | ティア混合時にLegendary一択になりがち | Legendaryカードは他2枚より少し小さく表示する等のUI工夫で「あえてRareを取る」選択を促す。または同ティア3枚の確率を上げる |

## 6. 役割別バフ適性マトリクス

バフカードと増強が各クラスでどの程度有用かの一覧:

| バフ/増強 | 총잡이(メイン推奨) | 물리딜러(メイン可) | 궁수(タワー推奨) | 마법사(タワー推奨) | 성직자(城守) |
|---------|:---:|:---:|:---:|:---:|:---:|
| 攻撃力UP | S | S | - | - | - |
| 攻撃速度UP | S | A | - | - | - |
| 移動速度UP | A | A | - | - | - |
| 射程UP | B | C | - | - | - |
| タワー強化 | - | - | S | S | - |
| 城修復 | A | A | A | A | B |
| 城バリア | A | A | A | A | S |
| R-01連鎖弾 | S | A | - | - | - |
| R-02吸血衝動 | A | S | - | - | - |
| R-03爆発弾頭 | - | - | A | S | - |
| L-06究極砲台 | - | - | S | A | - |
| L-07魂の連鎖 | S | A | - | - | - |

※ S=最適、A=有用、B=条件付き有用、C=恩恵小、- =対象外

メインキャラのクラスによってバフの価値が変わる設計は適切に機能している。ただし、タワーにセットしたクラスへの恩恵が「タワー強化」系に集約されており、「궁수タワーだけを強化する」といった個別強化バフは存在しない。これは現時点ではシンプルさを保つために問題ないが、将来の拡張余地として記録しておく。

## 7. 結論

### 良い点
1. バフカード8種の基本設計は堅実で、攻撃/防御/ユーティリティのバランスが取れている
2. 増強34種のティア差（Common/Rare/Legendary）は効果規模で明確に差別化されている
3. 組み合わせシナジー（特にL-07+L-06、L-02+R-06）がビルド構築の面白さを生む
4. バフカードの3択構成ロジック（カテゴリ分散）が「悩む選択」を促進する

### 改善が必要な点
1. **R-08の実装バグ**（通常モンスターダメージ0化）は早急に修正が必要
2. **C-06/C-12の実装不一致**（名前と実際の効果が異なる）
3. **L-04のLegendary感不足**（3秒間限定効果はLegendaryにふさわしくない）
4. **C-08/C-11/C-13/C-14の体感薄さ**（条件依存で恩恵ゼロになるケースが多い）
5. **属性マスタリー系の出現条件不備**（属性未取得時に効果ゼロ）

上記の調整提案を実施することで、全バフ/増強が「取る価値がある」状態になり、選択の面白さが一層向上する。
