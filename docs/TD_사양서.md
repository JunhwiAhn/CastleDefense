## 1-A. 종족 시스템

### 개요
- 첫 플레이 시 **종족 선택 화면** 등장 (1회 선택, 변경 불가)
- 종족에 따라 **인게임 패시브 보너스**, **타워 외관**, **가챠 풀 구성** 이 달라짐
- 가챠로 캐릭터를 해금 → 해금된 캐릭터를 타워 슬롯에 배치
- 미해금 캐릭터는 슬롯에 배치 불가 (자물쇠 표시)

### 가챠 × 종족 연동

| 항목 | 내용 |
|------|------|
| 기본 풀 | 자기 종족 캐릭터 70% + 타 종족 캐릭터 30% |
| 타 종족 캐릭터 | 해금 후 배치 가능 (패시브 보너스는 적용 안 됨) |
| 캐릭터 등급 | C / B / A / S (S가 최고, 종족 고유 캐릭터는 A~S) |
| 중복 해금 | 동일 캐릭터 중복 시 강화 재료로 전환 (각성 시스템 추후 구현) |

---

### 타워 배치 × 캐릭터 해금

```
타워 슬롯에 배치하려면:
  1. 해당 타입(궁수/대포/마법사/저격)에 맞는 캐릭터를 1명 이상 해금
  2. 슬롯 탭 → 해금된 캐릭터 목록에서 선택 → 배치

배치 시:
  - 타워 외관 = 배치한 캐릭터 스프라이트
  - 스탯 = 타워 기본 스탯 + 캐릭터 등급 보정
    (C: +0%, B: +5%, A: +10%, S: +15%)
  - 타워 이름 = "캐릭터이름의 진지" (예: "아리엘의 저격대")
```

### 3-2. 업그레이드 (3레벨)

- 레벨 1 → 2: 기본 비용 × 1.5, 스탯 +60%
- 레벨 2 → 3: 기본 비용 × 2.5, 스탯 +130% (레벨1 대비)
- 타워 위 탭 → 업그레이드/철거 팝업 표시
- 철거: 구매가의 50% 환급

### 3-3. 타겟 우선순위 (기본값: 경로 진행도 가장 높은 적)

변경 가능 (탭으로 전환):
1. **First** — 경로 진행도 가장 높음 (기본)
2. **Strongest** — HP 가장 높음
3. **Weakest** — HP 가장 낮음
4. **Closest** — 타워에서 거리 가장 가까움

### 5-4. 캐릭터 카드 업그레이드 시스템

#### 카드 레벨 (Lv 1~5)

| 레벨 | 필요 별조각 | 필요 카드 중복본 | 타워 스탯 보정 (등급별) |
|------|-----------|---------------|----------------------|
| Lv1 | — | — | C:+0% / B:+5% / A:+10% / S:+15% |
| Lv2 | 10 | 1장 | C:+4% / B:+9% / A:+14% / S:+20% |
| Lv3 | 25 | 2장 | C:+8% / B:+13% / A:+18% / S:+25% |
| Lv4 | 50 | 3장 | C:+12% / B:+17% / A:+22% / S:+30% |
| Lv5 | 100 | 5장 | C:+16% / B:+21% / A:+26% / S:+35% |

> **카드 중복본**: 가챠에서 이미 해금된 캐릭터를 다시 뽑으면 중복본 +1

#### 업그레이드 UI 흐름
```
홈화면 → [컬렉션] 탭 → 캐릭터 카드 선택
  → 카드 상세 팝업
      · 현재 레벨 / 다음 레벨 스탯 미리보기
      · 보유 별조각: 35 / 필요: 25 ⭐
      · 보유 중복본: 2 / 필요: 2장
      · [업그레이드] 버튼 (조건 미달 시 비활성)
```

#### 스탯 보정 적용 방식
```
실제 타워 데미지 = 기본 타워 데미지 × (1 + 등급보정 + 카드레벨보정)

예) S등급 카드 Lv3 궁수 배치 시:
  기본 데미지 8 × (1 + 0.25) = 10 데미지
```

#### 카드 중복본 대체 수단
- 별조각 30개로 중복본 1장 구매 가능 (홈 → 상점 → 카드 재료)
- 단, 동일 등급 카드만 구매 가능 (C중복본은 별조각 30, S중복본은 별조각 120)

| HP 표시 | 상단 HUD에 하트 아이콘 × HP |
| 게임오버 | HP = 0 |

> 기존 코드의 200HP 대미지 방식 → **적 1마리 = 성 HP 1 감소** 방식으로 변경

## 8. 게임 상태(State) 흐름

```
loading
  → home          (기존 홈화면 유지)
  → stageSelect   (스테이지 선택 — 1스테이지만 우선 구현)
  → prep          (웨이브 시작 전, 타워 배치 가능)
  → waving        (웨이브 진행 중)
  → waveCleared   (웨이브 클리어 인터벌 2초)
  → gameOver      (성 HP = 0)
  → stageClear    (전 라운드 클리어)
```
