# 属性システム仕様書

## 概要
5属性(火・水・地・電気・闇)による相性ボーナスダメージシステム。
キャラクター・モンスター双方に属性を付与し、戦略的な編成を促す。

## 相性表 (五行相克)

```
火 → 地 (焼き尽くす)
地 → 電気 (アース/接地)
電気 → 水 (感電)
水 → 火 (消火)
闇 ↔ 全属性に微弱ボーナス (中立・特殊)
```

| 攻撃属性 \ 防御属性 | 火 | 水 | 地 | 電気 | 闇 |
|---------------------|-----|-----|-----|------|-----|
| 火 | x1.0 | x0.75 | **x1.5** | x1.0 | x1.1 |
| 水 | **x1.5** | x1.0 | x0.75 | x1.0 | x1.1 |
| 地 | x0.75 | x1.0 | x1.0 | **x1.5** | x1.1 |
| 電気 | x1.0 | **x1.5** | x0.75 | x1.0 | x1.1 |
| 闇 | x1.1 | x1.1 | x1.1 | x1.1 | x1.0 |

- **有利**: x1.5 (50%ボーナスダメージ)
- **不利**: x0.75 (25%ダメージ減少)
- **闇**: 全属性に対してx1.1 (10%ボーナス)、対闇は等倍
- **無属性モンスター**: 全属性からx1.0 (等倍)

### 無属性キャラの属性付与
- レベルアップバフカードに「属性付与」カードを追加
- 例:「火炎の加護」→ メインキャラに火属性を付与 (ステージ中のみ)
- 1キャラ1属性のみ (後から付与すると上書き)

## 実装タスク

### Enum追加 (character_enums.dart)
```dart
enum ElementType {
  fire,    // 화염
  water,   // 수빙
  earth,   // 대지
  electric, // 번개
  dark,    // 암흑
  none,    // 무속성
}
```

### CharacterDefinitionに属性フィールド追加
```dart
final ElementType element; // デフォルト: ElementType.none
```

### モンスターに属性フィールド追加
```dart
// _Monsterクラスに追加
ElementType element; // デフォルト: ElementType.none
```

### ダメージ計算に属性倍率適用
```dart
double getElementMultiplier(ElementType attacker, ElementType defender) {
  // 相性表に基づく倍率を返す
}
```

### UI表示
- キャラアイコン横に属性アイコン表示
- ダメージ数字の色を属性色に変更
- 有利時「SUPER!」、不利時「RESIST」テキスト表示
