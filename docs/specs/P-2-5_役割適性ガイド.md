## メイン操作 vs タワー配置: 適性比較

### 궁수 (弓使い)

| | メイン | タワー |
|-|--------|--------|
| 射程 | 375px (比較的短い) | 375px |
| 3発同時 | 敵に向かって扇状3発 | 城に近い敵に扇状3発 |
| 操作感 | スティックで近づいて攻撃。機動力必要 | **最適**: ターゲット切り替えが速く常に城を守れる |
| 推奨配置 | タワーが理想。メインも可 | **タワー推奨 ★★★★★** |
| 弱点 | 射程が短いため敵に近づく必要 | タワー位置から遠い敵は届かない場合あり |

### 총잡이 (ガンスリンガー)

| | メイン | タワー |
|-|--------|--------|
| 5発バースト | 高DPS。連続ヒットが爽快 | 城に近い敵に連射 |
| 操作感 | メインとして爽快感が高い。**メイン推奨** | タワーも機能する |
| 推奨配置 | **メイン推奨 ★★★★★** | タワー可 ★★★ |
| 強み | バーストによる単体高DPS | 集中砲火で敵を素早く排除 |
| 弱み | 対多数が苦手（単体特化） | 複数の敵が来ると追いつかない |

### 마법사 (魔法使い)

| | メイン | タワー |
|-|--------|--------|
| 射程 | 600px (最長) | 600px |
| スプラッシュ | 範囲50px。密集した敵に有効 | 城に向かう敵の群れに対して有効 |
| 操作感 | 長射程で安全に攻撃できる | **最適**: 広いスプラッシュで複数処理 |
| 推奨配置 | メインも可 ★★★★ | **タワー推奨 ★★★★★** |
| 強み | 範囲攻撃で複数処理 | 敵の密集地点を効率的に掃討 |
| 弱み | 単体へのダメージが低め | — |

### 성직자 (聖職者)

| | メイン | タワー |
|-|--------|--------|
| 回復対象 | **メイン配置時**: メインキャラHPを回復 | **タワー配置時**: 城HPを5秒毎3回復 |
| 攻撃力 | なし | なし |
| 推奨配置 | 城HP危機時はタワー推奨 | **状況依存** |
| 戦略 | メイン配置: 生存重視プレイ向け | タワー配置: 長期戦向け (城修復) |
| 弱点 | 攻撃できないため1スロット分のDPSが失われる | |
| 推奨 | **1体以上タワーに配置推奨 ★★★★** | |

### 물리딜러 (近距離アタッカー)

| | メイン | タワー |
|-|--------|--------|
| 射程 | 525px | 525px |
| 攻撃タイプ | 近接（短射程の高ダメージ） | 城に近づいた敵を近距離で攻撃 |
| 操作感 | **メイン向き**: 積極的に敵に突撃するスタイル | タワーとしても機能するが、射程が短くカバー率が下がる |
| 推奨配置 | **メイン推奨 ★★★★** | タワー可 ★★ |
| 強み | 高単体ダメージ、近接で敵を素早く倒す | 城に到達した敵を素早く排除 |
| 弱み | 射程が短いため、敵との接触ダメージを受けやすい | タワー位置から遠い敵は無視しがち |

## 最適構成例

### 攻撃重視構成 (City Defense)
```
メイン:  총잡이 (高DPS、プレイヤー操作が活かされる)
T1(左上): 마법사 (範囲攻撃で群れを処理)
T2(右上): 궁수 (単体をピンポイントで排除)
T3(左下): 마법사 (反対側の群れを処理)
T4(右下): 궁수 (単体を排除)
```

### 長期戦/城防衛構成 (Castle Focus)
```
メイン:  총잡이 または 물리딜러
T1(左上): 성직자 (城HP継続回復)
T2(右上): 마법사 (範囲攻撃)
T3(左下): 궁수 (単体処理)
T4(右下): 마법사 (範囲攻撃)
```

### 初心者向け構成
```
メイン:  총잡이 (操作が分かりやすい)
T1〜T4: 궁수 × 4 (安定したDPS)
→ 聖職者不在だが、타워バフで十分カバー可能
```

## Engineerへのメモ

- タワーの攻撃ターゲット: **城に最も近い敵** (`_findMonsterNearestToCastle()` 活用)
- メインキャラの攻撃ターゲット: **最も近い敵** (既存ロジック維持)
- 聖職者のメイン/タワー判定: `partySlots[0]` が聖職者かどうかで分岐
