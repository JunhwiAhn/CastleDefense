## 城への接触ダメージ設計の根拠

### シミュレーション: 城HP=200が何体で尽きるか

**通常モンスターの場合:**
- 200体が城に到達 → ゲームオーバー
- スポーン間隔2秒 × 8体(R1) → 1ラウンドで全体が到達しても8ダメージ
- タワーが機能していれば 1〜2体が到達程度 → 実質ダメージは僅か

**ミニボスの場合:**
- 1体が居座って10秒ダメージを受けると: 2ダメ × (10秒÷1.5秒) ≈ 13ダメ
- 3体が同時に居座ると: 39ダメ/10秒 → 200HPを約50秒で溶かす
- タワーで素早く倒すことが重要な設計

**ボスの場合:**
- 1体が20秒間居座ると: 3ダメ × (20秒÷1.0秒) = 60ダメ
- 城HPの30%を1体のボスで削れる → 必殺技やタワーでの速攻が重要

## Engineer向け実装メモ

```dart
// 敵タイプ定義
enum MonsterType { normal, miniboss, boss }

// ステータス取得
MonsterStats getStats(MonsterType type, int round) {
  switch (type) {
    case MonsterType.normal:
      return MonsterStats(
        hp: _scaleHP(5, round),
        speed: min(60.0, 40.0 * _speedMult(round)),
        castleDmg: 1,
        castleDmgInterval: 0, // instant death (自爆)
        xpDrop: 1,
        goldDrop: 1,
        skillGauge: 0.03,
      );
    case MonsterType.miniboss:
      return MonsterStats(
        hp: _scaleHP(40, round),
        speed: min(50.0, 30.0 * _speedMult(round)),
        castleDmg: 2,
        castleDmgInterval: 1.5, // seconds (居座り型)
        xpDrop: 10,
        goldDrop: 5,
        skillGauge: 0.15,
      );
    case MonsterType.boss:
      return MonsterStats(
        hp: _scaleHP(200, round),
        speed: min(40.0, 25.0 * _speedMult(round)),
        castleDmg: 3,
        castleDmgInterval: 1.0, // seconds (居座り型)
        xpDrop: 50,
        goldDrop: 20,
        skillGauge: 0.50,
      );
  }
}
```
