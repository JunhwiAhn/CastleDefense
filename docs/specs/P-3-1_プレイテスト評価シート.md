## A. 操作感チェックリスト

### バーチャルスティック
- [ ] スティックの反応遅延が気にならない (1フレーム以内)
- [ ] デッドゾーンが適切 (意図しない微動がない)
- [ ] スティックを少し倒すと遅く動く (アナログ感度が機能している)
- [ ] スティックを全倒しで最大速度になる
- [ ] 指を離したら即停止する
- [ ] 方向転換がスムーズ (引っかかり感なし)

### 自動攻撃
- [ ] 最も近い敵に自動で攻撃する
- [ ] 攻撃の視覚フィードバックが分かりやすい (投射物が見える)
- [ ] 攻撃判定が適切 (当たっているのに当たらない/当たっていないのに当たる問題なし)

## B. 難易度チェックリスト

### ステージ1
- [ ] 初プレイでクリアできる
- [ ] 城HPが100以上残ってクリアできる
- [ ] レベルアップ選択画面が2〜3回表示される

### ステージ3〜5
- [ ] ミニボスが脅威として機能している
- [ ] タワー構成を考えなければクリアが難しい
- [ ] ボス戦が緊張感のある体験になっている

### 全体難易度感
- [ ] 「難しすぎる」「やめたい」と感じない
- [ ] 「もう1回やりたい」と感じる
- [ ] 死因が理解できる (何のせいで負けたか分かる)

## C. テンポチェックリスト

### ゲームのリズム
- [ ] ラウンド間3秒インターバルが短すぎず長すぎない
- [ ] XPジェムを回収しに行く動機がある (インターバル中に動いた)
- [ ] ボス前の緊張感がある

### レベルアップ
- [ ] 一時停止が不快でない
- [ ] 3択の選択に1〜5秒程度かかる (考える面白さがある)
- [ ] 選択後のゲーム再開がスムーズ

### ラウンドクリア演出
- [ ] 「ROUND CLEAR!」が分かりやすく表示される
- [ ] XP/ゴールド獲得の数字が確認できる

## D. 爽快感チェックリスト

### 攻撃の爽快感
- [ ] ヒットスパークVFXが気持ちいい
- [ ] 敵の死亡エフェクトが分かりやすい
- [ ] 大量の敵を一掃したときの達成感がある

### 必殺技 (スキルボタン)
- [ ] ゲージが100%になったことが分かる (視覚/音)

### 成長実感
