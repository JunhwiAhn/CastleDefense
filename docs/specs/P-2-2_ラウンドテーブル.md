## 同時上限(40体)による実際のスポーン挙動

R15以降の理論的なスポーン数(50体超)は40体上限でキャップされる。
→ キュー管理でオーバーフロー分は次タイミングにスポーン

### 実装方針（Engineerへ）
```dart
// スポーンキューの例
int _pendingSpawnCount = 0; // 待機中スポーン数

void _trySpawnFromQueue() {
  while (_pendingSpawnCount > 0 && _monsters.length < 40) {
    _spawnMonster(_currentRoundMonsterType);
    _pendingSpawnCount--;
  }
}
```
